# dependencies

## pygame

```
python -m pip install -U pygame --user
```

## requests

```
python -m pip install -U requests --user
```

## feedparser

```
python -m pip install -U feedparser --user
```

## pyinstaller (for build executable)

```
python -m pip install -U pyinstaller
```
